Publicar trabajo independiente

Esta vía está pensada para autores que quieran compartir papers, informes o escritos técnicos. Curamos entradas para correlacionarlas con países, temas y fuentes. No almacenamos archivos; comparte un enlace público (PDF o sitio web) y los metadatos básicos.

Enviar vía formulario

Si tenemos un endpoint activo (n8n), tu envío será procesado automáticamente.

Enviar vía Pull Request (recomendado)

Publica un archivo JSON con tus metadatos en nuestro repositorio público. Esto deja trazabilidad y versión pública.

  1. Visita el repo: github.com/vulcanoai/vulcanoai.github.io
  2. Crea un archivo en submissions/independiente/AAAA-MM/tu-trabajo.json con este esquema:
{
  "tipo": "independiente",
  "title": "…",
  "author": "…",
  "country": "México|Colombia|Regional",
  "topics": ["…"],
  "url": "https://…",
  "summary": "…",
  "license": "CC BY|CC BY-SA|…"
}

Abre un Pull Request. Nuestro pipeline de agentes validará y lo integrará al índice.
